# Rotoscoping
누크에서 로토스코핑을 이용해서 요소를 배경과 분리 할 수 있다.
외곽선에 유의하며 변하는 부분들을 생각해서 영역 나누기.
작업시 웬만하면 각 포인트 움직이지 X, 크기조절을 통해 모양 맞추기.
<br/> >Ctrl+Shift+Alt- 한점을 기준으로 크기조절
<br/> >Ctrl- 대칭 조절
<br/> >Shift- 자유변형
